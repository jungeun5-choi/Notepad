## 프로젝트 개요
### 📛 프로젝트명
> Java 메모장 기능 구현 프로젝트

### 🥅 달성하고자 하는 목표 
> 해당 프로젝트는 메인 과제인 [Naeil-nolja(호텔 예약 프로그램)](https://github.com/jungeun5-choi/Naeil-nolja)의 선수 과제이다.<br>
> 
> 메모장 기능을 구현하면서 메인 과제의 전체적인 흐름을 익힌다. 더불어 Java 언어와 협업에 익숙해지는 것을 목표로 한다. 팀원들은 각자 기능 단위로 나누어 일을 맡는다.<br>
